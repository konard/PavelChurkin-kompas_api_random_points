# kompas_api_random_points

Приложение для КОМПАС-3D, которое случайным образом размещает на чертеже пары соосных кругов (кольца) с чёрной заливкой между контурами.

## Описание

При запуске приложение открывает окно настройки со следующими параметрами:
- **Количество пар кругов** — сколько кольцевых фигур разместить на листе
- **Диаметр внешнего круга** (мм) — диаметр внешнего контура
- **Диаметр внутреннего круга** (мм) — диаметр внутреннего контура
- **Минимальное/максимальное расстояние между центрами** (мм) — диапазон расстояний между центрами пар кругов (по умолчанию 50–80 мм)
- **Формат листа** — A0, A1, A2, A3, A4
- **Ориентация** — книжная или альбомная
- **Количество листов** — при добавлении нового листа создаётся новое случайное расположение с оформлением «без рамок»

После нажатия кнопки «Draw» приложение подключается к КОМПАС-3D и создаёт чертёж с заданными параметрами.

## Требования

- **ОС**: Windows (КОМПАС-3D работает только на Windows)
- **КОМПАС-3D**: версия 16 или новее (установлена и активирована)
- **Python**: 3.7+
- **pywin32**: для COM-автоматизации

## Как работает приложение

1. **GUI (tkinter)** — открывается окно настройки параметров
2. **Подключение к КОМПАС-3D** — через COM-автоматизацию (API5 + API7)
3. **Создание документа** — новый чертёж с оформлением «без рамок» (`layoutName = ""`)
4. **Генерация позиций** — случайное размещение в пределах листа с учётом минимального расстояния между центрами
5. **Рисование** — для каждой пары: два соосных круга + штриховка (сплошная чёрная заливка) между контурами
6. **Многолистовой режим** — при необходимости добавляются новые листы, каждый с новым случайным расположением

## Подключение как плагин КОМПАС-3D (RTW / DLL)

### Формат RTW (КОМПАС-Макро)

КОМПАС-3D поддерживает макросы на Python через встроенную систему «КОМПАС-Макро»:

1. Откройте КОМПАС-3D → Сервис → Макросы → Макросы на Python
2. Создайте новый макрос или импортируйте `kompas_random_circles.py`
3. При работе через макро-среду доступны модули `Kompas6API5` и `KompasAPI7` напрямую (без `win32com`)
4. Макросы сохраняются в формате `.py` и могут быть запущены из меню КОМПАС

Для распространения макрос можно упаковать в `.rtw` файл — это архив с описанием и Python-скриптом, который КОМПАС загружает автоматически.

### Формат DLL (КОМПАС-Библиотеки)

Для создания прикладной библиотеки (DLL) на Python:

1. Используйте фреймворк `ctypes` или `cffi` для создания DLL-обёртки
2. Или используйте C++/C# обёртку, вызывающую Python через встроенный интерпретатор
3. Прикладные библиотеки КОМПАС размещаются в формате `.dll` / `.rtw` и подключаются через: Сервис → Менеджер библиотек → Добавить библиотеку

**Рекомендуемый подход**: использование Python-скрипта напрямую через COM-автоматизацию (как реализовано в данном проекте) — это наиболее простой и гибкий метод.

## Настройка и разработка в PyCharm

### Создание проекта

1. Откройте PyCharm → File → Open → выберите папку `kompas_api_random_points`
2. Настройте интерпретатор Python:
   - File → Settings → Project → Python Interpreter
   - Выберите системный Python (где установлен `pywin32`) или создайте виртуальное окружение
3. Установите зависимости: откройте Terminal в PyCharm и выполните:
   ```bash
   pip install -r requirements.txt
   ```

### Настройка запуска

1. Run → Edit Configurations → Add New → Python
2. **Script path**: укажите `kompas_random_circles.py`
3. **Parameters**: `--debug` (опционально, для подробного логирования)
4. **Working directory**: корень проекта
5. Нажмите Apply → OK

### Отладка

1. Установите точки останова (breakpoints) в коде
2. Запустите через Run → Debug
3. PyCharm остановится на breakpoint и позволит исследовать переменные

### Авто-дополнение для КОМПАС API

Для подсказок IDE по COM-объектам КОМПАС:

1. Запустите один раз `win32com.client.gencache.EnsureModule(...)` — это создаст Python-обёртки для COM-интерфейсов
2. Сгенерированные файлы появятся в `~\AppData\Local\Temp\gen_py\<python_version>\`
3. PyCharm будет использовать их для автодополнения

## API-справка

Приложение использует два уровня API КОМПАС-3D:

| API | Назначение | COM ProgID |
|-----|-----------|------------|
| API5 | Геометрия чертежа (круги, штриховка, контуры) | `KOMPAS.Application.5` |
| API7 | Управление документами (открытие, сохранение, листы) | `Kompas.Application.7` |

### Ключевые методы API5

- `ksCircle(xc, yc, rad, style)` — рисование круга
- `ksContour(draw)` / `ksEndObj()` — определение замкнутого контура
- `ksNewGroup(type)` / `ksEndGroup()` — группа границ для штриховки
- `ksHatch(hatchParam)` — применение штриховки/заливки
- `ksCreateDocument(docParam)` — создание нового документа
